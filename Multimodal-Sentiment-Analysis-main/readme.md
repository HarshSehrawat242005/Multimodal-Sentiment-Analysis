# Multimodal Sentiment Analysis
This project implements sentiment analysis using text, image, and audio data.

## Directory Structure
- `data_preprocessing.py`: Handles data preparation.
- `data_visualization.py`: Visualizes dataset.
- `model_creation.py`: Builds and trains a multimodal model.
- `model_testing.py`: Tests the model.
- `pipeline.py`: Automates the workflow.

## Installation
Run `pip install -r requirements.txt` to install dependencies.

## Deployment
Run `python deployment/app.py` to start the API.
